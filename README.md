# 📱 D-App

Aplikasi **Decentralized App (D-App)** berbasis web. Proyek ini dirancang untuk mengintegrasikan konsep blockchain atau teknologi terdesentralisasi.

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
